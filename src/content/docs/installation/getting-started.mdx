## Introduction

Ce guide permet de démarrer avec Plac. Il explique comment installer les prérequis avant d'utiliser Plac.

Plac est une application de suivi de chantier pour les artisans du bâtiment.

## Prérequis

<Aside>
Pour utiliser ce projet vous devrez disposer de :

- `php` >=8.2
- `mariadb` >=10.11
- `Composer`
- `Yarn` ou `Npm`
</Aside>


## Deploy

À savoir que le deployement de Plac est géré automatiquement, il suffit de `push` sur la branche `dev`| `prod` pour que les modifications soient prises en compte.
