Plac API est une API RESTful qui permet de gérer la data pour Plac. Cette API est utilisée par l'application web `Admin Plac` pour gérer les boutiques. Elle est aussi utilisé par le `Bundle Plac` qui permet à chacune des boutiques de récupérer les informations qui la concerne.
