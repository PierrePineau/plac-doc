## API

<Steps>

1. Cloner le projet à partir de Github

2. Installer les dépendances du projet avec `Composer`
    ```bash
    composer install
    ```
3. Générer les clées encryptages utilisées par `JWT`
    ```bash
    php bin/console lexik:jwt:generate-keypair
    ``` 
3. Vous <b>devez</b> créer un fichier `.env.local` à la racine du projet pour définir des paramètres uniquement pour vous en local
    ```bash
    APP_ENV=dev
    TRUSTED_IPS=""
    APP_DEV_MODE=true
    DATABASE_URL="mysql://{username}:{password}@{host}:{port}/plac-dev"
    ...
    ```
    
    <LinkCard
        title="Configuration API"
        description="Consultez la page de configuration de l'API pour plus d'information"
        href="/api/configuration.mdx"
    />
</Steps>
