## Commandes Utiles

Créer une `Entity` avec le maker de Symfony.

```bash
    php bin/console make:Entity EntityName
```

Pour en savoir plus sur le format des BDD.s, vous pouvez consulter les schémas :
- [Diagrams](./Diagrams.mdx)

### Update DBB

Pour mettre à jour la BDD, vous pouvez utiliser la commande suivante :

```bash
    php bin/console make:migration
```

Puis pour appliquer les migrations :

```bash
    php bin/console doctrine:migrations:migrate
```
